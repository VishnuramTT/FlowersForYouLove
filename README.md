# FlowersForYouLove

A tiny GitHub Pages–friendly static page.

## Run locally

### Option 1 (no installs)

Open `index.html` in your browser.

### Option 2 (recommended: any static server)

From the repo root, run one of these:

```bash
# If you have Python 3 installed
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish on GitHub Pages

Repo: `VishnuramTT/FlowersForYouLove`

1. GitHub → Repo **Settings** → **Pages**
2. **Deploy from a branch**
3. Branch: `main`
4. Folder: `/ (root)`

Your site will be available at:

`https://vishnuramtt.github.io/FlowersForYouLove/`
